<br/>
偶然在网上找到了第一次去西藏的照片，记得上次回家的时候家里的电脑坏了，本来想翻翻老照片的，由于没时间修电脑也只能作罢，
西藏是每个行者梦想的远方，一尘不染的仙境，尤其无论从青藏，川藏，滇臧，新藏各个路线进藏都充满了坎坷和艰辛，去西藏好比是一场去往天堂的朝圣之路抑或是心灵净化之路，又或是一场成人礼，无关乎信仰或是什么，这一路上所见到的人或事都足以让任何人铭记一生。
<br/>
{% img /images/tibet/2.jpg 400 400 %}
{% img /images/tibet/4.jpg 400 400 %}
<br/>
世界上到处都有很美的地方，高山湖泊，大海沙滩，甚至于极光，但西藏的美是特别的，每一座山脚下都能碰到虔诚的徒步转山人，每一座寺庙都会看到步步朝拜之人。这种景象给人的震撼并不仅仅是自然的力量。尤其在高原状况下，时常会出现一种虚幻缺氧的状态更加加深了对每一个神山圣湖的畏惧和敬仰。
<br/>
{% img /images/tibet/3.jpg 400 400 %}
{% img /images/tibet/5.jpg 400 400 %}
<br/>

一开始并不理解那些骑行或者徒步的人的心情，很难想象一路上要翻越十几座四，五千米高的雪山，但渐渐地旅行多了，我开始能慢慢地懂那些人了。一次这样的旅行能给一个人的生命带来怎样的光彩，这是那些躲在舒适区的人所想象不到的。即使幸苦，但当你到达目的地那一刻的释放高潮是无与伦比的。
<br/>
{% img /images/tibet/8.jpg 400 400 %}
{% img /images/tibet/7.jpg 400 400 %}
<br/>
出生在和平年代的我们，一生能经历的高潮低谷是很有限的，但这种苦行僧般的旅行过程中给人以难以想象的大起大落的快感也许正是我们这一代人所缺乏的。
在一生中，也许每个人都应该找个合适的时间，放下现实生活的不堪，以西藏为目的，无畏地旅行一次，做一场梦。
<br/>
{% img /images/tibet/6.jpg 800 400 %}
<br/>
过了那么久，路上的幸苦抑或是美好其实已经不记得了，留下的只有面对西藏每一点每一滴那种刻进骨髓里的震撼。
